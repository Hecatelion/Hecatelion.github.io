Pitch, Context & Trivia

Exercise in which I had to act like answering a request for proposal of which topic was 'make an adaptation of the boar game Pandemic : Rapid Response'. During this project I tried to focus in a first time on main features, I tried to use the original game assets in order to make clients reassured about the game environement. Then when the prototype was fully playable, I wrote a presentation document explaining my choices and giving some ideas about this prototype possible evolution. I especially gave 2 versions, 2 visions of what could be a Pandemic adaptation depending on the marketing target and intentions.

What did I do?

As Programmer

  • analyse and understand in depth game features
  • task priorisation
  • implementing all game rules and mecanics
  • turn and time system
  • pawn, dice, crates manipulation system
  • card drawing and task completion
  • option menu
  • win/lose condition

As Designer

  • getting the desired experience provided by the game design of Pandemic
  • adatpting this physical game experience to a virtual game
  • thinking about the client intentions and possible marketing
    to make relevant decision about my design
  • suggest 2 design approaches :
    - management game (with strong video game identity)
    - board game (with physical pawn manipulation)
  • making a presentation document to convince and explain my concepts
    and choices to the clients
