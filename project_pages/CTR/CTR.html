Pitch

A Cut The Rope reproduction in a Pokemon theme. Developped in order to improve prototyping and level designing skills. Indeed 10 new levels has been designed is which I tried to make level finishing easy but getting all stars, hard.

What did I do?

As Programmer

  • all gameplay programming
  • rope physics using unity joints
  • save system
  • sound effects and music management

As Level Designer

  • making level fitting with an existing game
  • 10 levels created
  • levels with multiple completion ways
  • different difficulty path on same levels
